# Third Party Notices

## WebGL rain glass renderer inspiration

The WebGL rain wallpaper renderer is an original implementation inspired by:

- Codrops RainEffect by Lucas Bebber: https://github.com/codrops/RainEffect
- Steam Workshop "Drops" visual reference: https://steamcommunity.com/sharedfiles/filedetails/?id=955880074

No Steam Workshop assets or source files are bundled in LiveWallpaper. The
renderer uses app-generated JavaScript, canvas water maps, and WebGL shaders.
