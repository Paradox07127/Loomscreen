## License

LiveWallpaper is provided for personal desktop customization on macOS, subject to the license terms selected for the public release.

## User Content

Users are responsible for the videos, HTML files, folders, remote URLs, Wallpaper Engine packages, and other content they choose to use with LiveWallpaper. Users should only use content they have the right to access and display.

## Remote HTML Content

Remote HTML wallpapers are user-selected web content. LiveWallpaper can disable JavaScript for untrusted origins and provides controls for tracker blocking and ephemeral browsing data, but the user remains responsible for the URLs they choose to load.

## Local File Access

LiveWallpaper relies on macOS file picker permissions and security-scoped bookmarks to access selected local files and folders after relaunch. If access expires or files are moved or deleted, the user may need to reselect the content.

## Weather And Location

Weather-reactive effects depend on third-party weather and optional location services. Weather, location, and network availability may be inaccurate, delayed, unavailable, or denied by system settings.

## Performance And Battery

Live wallpapers can use CPU, GPU, memory, and battery power. Users should adjust frame rate, pause, battery, full-screen, and effect settings based on their device and workload.
